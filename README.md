# TrapScan Backend
## Setup
TrapScan backend uses Laravel 8.

You can setup a dev environment using Docker with Laravel Sail.

1) Install Docker
2) Clone this repository
3) Run ``sail artisan migrate:fresh --seed`` to setup and populate the database
